# Number Hunt
Shooting range game for AVR-MT128


## Requirements
[Ubuntu 24](https://ubuntu.com/desktop) or newer


## Running the program

###

Install the required packages and clone the [simavr](https://github.com/akosthekiss/simavr.git) repository:
```
bash install.sh
```

Build the project:
```
bash build.sh
```

Run the game:
```
bash run.sh
```


## Controls

| Button   | Action        |
|----------|:-------------:|
| Number 5 | Start / Shoot |
| Number 8 | Move Up       |
| Number 4 | Move Left     |
| Number 6 | Move Right    |
| Number 2 | Move Down     |

*Tip: use the **numeric keypad** for the best experience (Num Lock must be enabled).*


## Gameplay
Try to shoot the numbered targets in ascending order before the time runs out. Missed shots will reduce your HP by 1.
